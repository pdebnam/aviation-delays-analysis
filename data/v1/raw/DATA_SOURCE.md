
## Raw Data Source

The raw flight data used in this project comes from the
U.S. Bureau of Transportation Statistics (BTS)
"Reporting Carrier On-Time Performance" dataset.

Due to file size limitations, the raw CSV files are not
stored in this repository.

### How to obtain the data
1. Visit: https://www.transtats.bts.gov/
2. Select "Reporting Carrier On-Time Performance"
3. Download the desired month and year
4. Save the CSV file locally in `data/raw/`
